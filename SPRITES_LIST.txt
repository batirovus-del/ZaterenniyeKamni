=== ИНСТРУКЦИЯ ПО ЗАМЕНЕ СПРАЙТОВ ===
================================================================================

СПОСОБ 1: Через Unity Inspector (рекомендуется для отдельных элементов)
------------------------------------------------------------------------
1. Открой Unity проект ZateryannieKamni
2. Открой сцену: Assets/Scenes/Lobby.unity или Game.unity
3. В окне Hierarchy найди UI элемент (например, кнопку)
4. Кликни на него
5. В окне Inspector найди компонент "Image" или "SpriteRenderer"
6. Найди поле "Source Image" (или "Sprite")
7. Кликни на кружок справа от поля
8. Выбери новый спрайт из списка
9. Готово!

СПОСОБ 2: Замена файла (для массовой замены)
----------------------------------------------
1. Найди оригинальный спрайт в папке:
   C:\Users\bayra\Documents\Games\ZateryannieKamni\Assets\Sprites\
   
2. Запомни его ТОЧНОЕ имя (например: Btn_Normal.png)

3. Подготовь новый спрайт:
   - Сохрани с ТАКИМ ЖЕ ИМЕНЕМ
   - Формат: PNG с прозрачностью
   - Размер: примерно как у оригинала

4. Удали старый файл из Unity (Delete в Project окне)

5. Перетащи новый файл в ту же папку

6. Unity автоматически обновит все ссылки!

СПОСОБ 3: Программная замена (через код)
-----------------------------------------
using UnityEngine;
using UnityEngine.UI;

public class ChangeSprite : MonoBehaviour
{
    public Image targetImage;
    public Sprite newSprite;
    
    void Start()
    {
        targetImage.sprite = newSprite;
    }
}

НАСТРОЙКА ИМПОРТА СПРАЙТА
--------------------------
После добавления нового спрайта:

1. Выбери спрайт в Project окне
2. В Inspector установи:
   - Texture Type: Sprite (2D and UI)
   - Sprite Mode: Single
   - Pixels Per Unit: 100
   - Filter Mode: Bilinear
   - Compression: Normal Quality
3. Нажми Apply

ДЛЯ РАСТЯГИВАЕМЫХ КНОПОК (9-Slice)
-----------------------------------
1. Выбери спрайт кнопки
2. Нажми кнопку "Sprite Editor"
3. Настрой зеленые границы (отступы от краев)
4. Нажми Apply
5. В компоненте Image установи "Image Type: Sliced"

ПОИСК ИСПОЛЬЗОВАНИЯ СПРАЙТА
----------------------------
Чтобы найти где используется спрайт:
1. Правый клик на спрайт в Project
2. Выбери "Find References In Scene"
3. Unity покажет все объекты с этим спрайтом

СОВЕТЫ
------
✓ Сохраняй размеры: новый спрайт должен быть примерно того же размера
✓ Используй PNG с прозрачностью для UI
✓ Разрешение для мобильных: 512x512 или 1024x1024
✓ Делай бэкап перед массовой заменой!
