# 🪨 Затерянные камни (Lost Stones)

**Match-3 головоломка** для [Яндекс.Игр](https://yandex.ru/games/). Собирай драгоценные камни, взрывай бомбы, проходи сотни уровней и спасай волшебных персонажей в этом увлекательном приключении!

## ✨ Особенности

- 🎮 Классический геймплей match-3 с уникальными механиками
- 💎 6 видов драгоценных камней и множество специальных фишек (бомбы, радуга, конфеты, хамелеоны)
- 🎯 9 режимов игры: Score, BringDown, Jelly, Digging, SweetRoad, Rescue и другие
- 🚀 Бустеры: молот, конфетный заряд, взрывы по горизонтали/вертикали
- 🏆 Таблица лидеров и прогрессия по 640+ уровням
- 🌍 Локализация: русский, английский
- 📱 Оптимизировано для WebGL (Yandex Games)

## 🛠 Технологии

- **Движок:** Unity
- **Платформа:** WebGL (Яндекс.Игры)
- **SDK:** PluginYG v1.6
- **2D-графика:** 2D Toolkit (tk2d)
- **Анимации:** DOTween
- **Локализация:** I2 Localization

## 🚀 Сборка

1. Открыть проект в Unity
2. Выбрать WebGL-шаблон: `1.6_(FixedRatio)`
3. File → Build Settings → WebGL → Build
4. Загрузить билд в кабинет разработчика Яндекс.Игр

## 📋 Требования Яндекс.Игр

- SDK v2 (PluginYG v1.6)
- Автоопределение языка (пункт 2.14)
- Показ рекламы: Fullscreen Interstitial, Rewarded Video
- Сохранение прогресса: Cloud Storage
- Покупки: Yandex Payments (IAP)
